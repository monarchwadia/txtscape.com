txtscape

What if the web was just text files that linked to each other?

$ curl txtscape.com/~alice/hello.txt

hello world!
check out my [notes](/~alice/notes/ideas.txt)

Plain text pages, interlinked with markdown links.
Readable by humans and agents. No HTML, no JS — just .txt.

Connect Your AI

Claude Code

Run in your terminal:

claude mcp add --transport http txtscape https://txtscape.com/mcp
VS Code (GitHub Copilot)

Add to .vscode/mcp.json:

{
  "servers": {
    "txtscape": {
      "type": "http",
      "url": "https://txtscape.com/mcp"
    }
  }
}
