txtscape

Build a Text Adventure with AI Memory

Your AI agent is powerful — but it forgets everything between conversations.

Every chat starts from zero. The agent re-reads your code, re-discovers your design, re-invents names it already chose yesterday. You end up repeating yourself, correcting inconsistencies, losing momentum.

What if your agent could remember?

In this tutorial, you'll build Shadows of Eldoria — a text adventure game in a single index.html file. But you won't write the game yourself. You'll describe your game world in txtscape — rooms, items, characters — and your AI agent will build the game from that knowledge.

What you'll learn

  • How to install and configure txtscape
  • How to store structured knowledge in pages
  • How the AI reads your pages to generate consistent code
  • How to organize pages into folders for different concerns
  • What happens when you change a page and regenerate — the AI adapts
  • How to search and cross-reference your project memory
Prerequisites
Node.js 18+ • VS Code with GitHub Copilot (or any MCP-compatible IDE) • ~30 minutes

Steps

  • 1. Setup — install txtscape and create your project
  • 2. Your First Room — describe a tavern in txtscape
  • 3. Summoning Code — the AI builds your game
  • 4. Expanding the World — rooms, items, characters
  • 5. The Living World — change a page, code adapts
  • 6. Mastery — search, history, what's next
