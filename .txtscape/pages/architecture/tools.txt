# MCP Server Tools

Total: 11 tools

## Core CRUD
- get_page: Read a page. Returns content + SHA-256 hash in _meta field for optimistic concurrency.
- put_page: Create/update a page. Optional expected_hash rejects stale writes. Returns _meta.hash of new content.
- append_page: Append to a page (or create). Optional expected_hash. Returns _meta.hash of new content.
- delete_page: Remove a page. Cleans up empty parent folders.
- move_page: Rename/relocate a page. Creates dest folders, cleans up empty source folders.

## Discovery
- list_pages: Browse directory tree with previews. Default: recursive (shows all files with full paths). Set recursive=false for shallow folder-only listing.
- search_pages: Full-text search across all pages. Supports isRegex=true.
- related_pages: Cross-reference graph — outgoing + incoming refs. Suffix matching (not just full path).

## Editing
- str_replace_page: Surgical edit — old_str must match exactly once. Returns unified-diff snippet with -/+ markers. Returns _meta.hash. Optional expected_hash.

## Bulk & History
- snapshot: Concatenate subtree with relative path headers. Summary (N pages, N bytes). Max 100 per call, offset for pagination.
- page_history: Git commit history for a page. include_diff=true for patches. limit (default 20).
