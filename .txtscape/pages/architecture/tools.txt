# MCP Server Tools

Total: 11 tools

## Core CRUD
- get_page: Read a page. Returns content + SHA-256 hash in _meta field for optimistic concurrency.
- put_page: Create/update a page. Optional expected_hash rejects stale writes.
- append_page: Append to a page (or create). Optional expected_hash. Atomic, no read-modify-write.
- delete_page: Remove a page. Cleans up empty parent folders.
- move_page: Rename/relocate a page. Creates dest folders, cleans up empty source folders.

## Discovery
- list_pages: Browse directory tree with previews. Supports recursive=true.
- search_pages: Full-text search across all pages. Supports isRegex=true.
- related_pages: Cross-reference graph — outgoing + incoming refs. Suffix matching (not just full path).

## Editing
- str_replace_page: Surgical edit — old_str must match exactly once. Returns diff snippet. Optional expected_hash.

## Bulk & History
- snapshot: Concatenate subtree with relative path headers. Summary (N pages, N bytes). Max 100 pages with truncation warning.
- page_history: Git commit history for a page. include_diff=true for patches. limit (default 20).
