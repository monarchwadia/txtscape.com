# MCP Server Tools

Total: 11 tools

## Core CRUD
- get_page: Read a page. Returns content + SHA-256 hash for optimistic concurrency.
- put_page: Create/update a page. Optional expected_hash rejects stale writes.
- append_page: Append to a page (or create). Atomic, no read-modify-write.
- delete_page: Remove a page. Cleans up empty parent folders.
- move_page: Rename/relocate a page. Creates dest folders, cleans up empty source folders.

## Discovery
- list_pages: Browse directory tree with previews. Supports recursive=true.
- search_pages: Full-text search across all pages. Supports isRegex=true.
- related_pages: Cross-reference graph — outgoing links + incoming references.

## Editing
- str_replace_page: Surgical edit — old_str must match exactly once.

## Bulk & History
- snapshot: Concatenate all pages in a subtree into one response.
- page_history: Git commit history for a page (requires git repo).
