# Test Helpers

Shared helpers for txtscape-mcp tests. Live in test files, not production code.

## MCP Helpers (txtscape-mcp)

  setupTestServer(t) *server           — creates server with temp dir + .txtscape/pages/
  callMethod(s, method, params) resp   — sends JSON-RPC request
  callTool(s, name, args) resp         — sends tools/call request
  getTextContent(t, resp) string       — extracts text from tool response
  isToolError(resp) bool               — checks if response is a tool error

## Usage Pattern

Helpers keep test bodies focused on the journey, not boilerplate:

  func TestJourney_FullLifecycle(t *testing.T) {
      s := setupTestServer(t)
      resp := callTool(s, "put_page", map[string]string{"path": "test.txt", "content": "hello"})
      text := getTextContent(t, resp)
      // assert on text
  }
